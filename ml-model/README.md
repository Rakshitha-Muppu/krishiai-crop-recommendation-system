# Machine Learning Model

This folder contains the machine learning models used in the project.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn

## Functions

* Data Preprocessing
* Crop Recommendation
* Yield Prediction

## Input

* Soil Type
* Rainfall
* Temperature
* Land Area
* Crop Information
* Processed Agricultural Dataset

## Output

* Recommended Crop
* Expected Yield

## Purpose

* Train machine learning models using agricultural data
* Analyze historical agricultural datasets
* Generate crop recommendations
* Predict expected crop yield
