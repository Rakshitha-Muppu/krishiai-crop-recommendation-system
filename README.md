# KrishiAI - Crop Recommendation System

## Project Overview

KrishiAI is a machine learning-based web application that recommends suitable crops based on environmental conditions and soil characteristics.

The application integrates a Flask backend, Random Forest machine learning model, MySQL database, weather API, and a responsive frontend to provide personalized crop recommendations.

---

## Problem Statement

Farmers often face difficulty selecting suitable crops based on changing environmental conditions.

KrishiAI helps users make data-driven crop selection decisions using machine learning predictions based on:

* Temperature
* Humidity
* Rainfall
* Soil Type

---
